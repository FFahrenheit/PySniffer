# PySniffer
Real time Network Sniffer using pure Python and winpcapy

## Supported modes:
- Read from .bin or .txt file
- Real time to console
- Real time to .txt log file

## Packets supported
- [x] Ethernet
- [x] IPv4
- [x] ICMPv4
- [x] ARP
- [x] RARP
- [x] IPv6
- [x] ICMPv6
- [x] TCP
- [x] UDP
- [x] DNS
